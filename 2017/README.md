## Initialize environments

- `brew install go` - install go
- `brew install entr` - run tests in watch mode
- `brew install gotestsum` - run tests with summary
- `make run-dev year=2017 day=01` - run day 1 of year 2017 in dev mode
- `make watch-test year=2017 day=01` - run test for day 1 of year 2017 in watch mode
